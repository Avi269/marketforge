# MarketForge

MarketForge is a full-stack multi-vendor e-commerce marketplace built to explore real-world software engineering practices using Django.

The platform allows customers to browse and purchase products from multiple vendors while providing vendors with tools to manage listings, monitor sales performance, track inventory, and interact with customer reviews.

The project is being developed with a strong focus on backend engineering, database design, authentication and authorization, API development, containerization, cloud deployment, and scalable software architecture.

---

## Planned Features

### Customer Features

* User registration and authentication
* Product browsing and search
* Category-based navigation
* Wishlist management
* Shopping cart and checkout
* Order tracking
* Product reviews for verified purchases

### Vendor Features

* Vendor application and approval workflow
* Product listing management
* Inventory tracking
* Sales dashboard and analytics
* Review response system
* Order management

### Administrative Features

* Vendor approval and moderation
* Product and category management
* Order oversight
* User management
* Marketplace analytics
